# ­ЪљЇ MiniPy
**рЃЏрЃАрЃБрЃЉрЃБрЃЦрЃў рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃбрЃўрЃБрЃџрЃў Python рЃњрЃљрЃарЃћрЃЏрЃЮ**
*рЃњрЃљрЃюрЃЎрЃБрЃЌрЃЋрЃюрЃўрЃџрЃўрЃљ рЃАрЃгрЃљрЃЋрЃџрЃћрЃЉрЃўрЃА рЃЊрЃљрЃАрЃљрЃгрЃДрЃўрЃАрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃљрЃгрЃДрЃЮрЃЉрЃўрЃџрЃўрЃљ PySide6 рЃЊрЃљ qtconsole рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃћрЃЉрЃќрЃћ.*

---

## ­Ъј» рЃЏрЃўрЃќрЃљрЃюрЃў
**MiniPy** рЃљрЃарЃўрЃА рЃърЃљрЃбрЃљрЃарЃљ, рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃљрЃЌрЃљрЃерЃЮрЃарЃўрЃАрЃў рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃА рЃбрЃћрЃЦрЃАрЃбрЃБрЃа рЃарЃћрЃЊрЃљрЃЦрЃбрЃЮрЃарЃАрЃљ рЃЊрЃљ рЃЕрЃљрЃерЃћрЃюрЃћрЃЉрЃБрЃџ Python REPL-рЃА (IPython-рЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃА).
рЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃљрЃФрЃџрЃћрЃЋрЃА рЃЏрЃЮрЃАрЃгрЃљрЃЋрЃџрЃћрЃћрЃЉрЃА рЃЊрЃљ рЃЊрЃљрЃЏрЃгрЃДрЃћрЃЉрЃЌ, рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃЊрЃљрЃгрЃћрЃарЃЮрЃю рЃЊрЃљ рЃњрЃљрЃљрЃерЃЋрЃЮрЃю Python рЃЎрЃЮрЃЊрЃў Рђћ рЃБрЃцрЃарЃЮ рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃбрЃўрЃБрЃџрЃљрЃЊ, рЃЋрЃўрЃЊрЃарЃћ IDLE, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃњрЃљрЃфрЃўрЃџрЃћрЃЉрЃўрЃЌ рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ, рЃЋрЃўрЃЊрЃарЃћ VS Code.

---

## ­Ъњ╗ рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃўрЃџрЃў рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃћрЃЉрЃў
- **Manjaro Linux** (Hyprland, GNOME, XFCE)
- **Windows 10/11**
- рЃАрЃ«рЃЋрЃљ **Linux рЃЊрЃћрЃАрЃЎрЃбрЃЮрЃърЃћрЃЉрЃў** (Wayland рЃЊрЃљ X11 рЃњрЃљрЃарЃћрЃЏрЃЮрЃћрЃЉрЃў)

---

## ­ЪДЕ рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃў
- рЃќрЃћрЃЊрЃљ **рЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃў рЃарЃћрЃЊрЃљрЃЦрЃбрЃЮрЃарЃў** (`QPlainTextEdit`)
- рЃЦрЃЋрЃћрЃЊрЃљ **REPL рЃцрЃљрЃюрЃ»рЃљрЃарЃљ** (IPython/qtconsole)
- рЃдрЃўрЃџрЃљрЃЎрЃћрЃЉрЃў:
  - ­ЪЌѓ№ИЈ **Open** Рђћ рЃцрЃљрЃўрЃџрЃўрЃА рЃњрЃљрЃ«рЃАрЃюрЃљ
  - ­ЪњЙ **Save** Рђћ рЃцрЃљрЃўрЃџрЃўрЃА рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ
  - РќХ№ИЈ **Run** Рђћ рЃАрЃЎрЃарЃўрЃърЃбрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ REPL-рЃерЃў
  - ­ЪД╣ **Clear** Рђћ REPL-рЃўрЃА рЃњрЃљрЃАрЃБрЃцрЃЌрЃљрЃЋрЃћрЃЉрЃљ
- рЃўрЃДрЃћрЃюрЃћрЃЉрЃА **рЃАрЃўрЃАрЃбрЃћрЃЏрЃБрЃа Python 3.12РђЊ3.13+ рЃўрЃюрЃбрЃћрЃарЃърЃарЃћрЃбрЃљрЃбрЃЮрЃарЃА**
- рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃарЃЮрЃњрЃЮрЃарЃф **Wayland**, рЃљрЃАрЃћрЃЋрЃћ **X11** рЃЊрЃљ **Windows** рЃњрЃљрЃарЃћрЃЏрЃЮрЃћрЃЉрЃерЃў
- рЃАрЃарЃБрЃџрЃўрЃљрЃЊ **рЃърЃЮрЃарЃбрЃљрЃЉрЃћрЃџрЃБрЃарЃўрЃљ** (PyInstaller рЃљрЃю AppImage рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃЌ)

---

## ­Ъџђ рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ рЃЊрЃљ рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ

### рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃћрЃЉрЃћрЃЉрЃў
```bash
sudo pacman -S python-pip python-jupyter_client python-ipykernel qt6-base
pip install --user PySide6 qtconsole
```
## ­ЪЊд рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃў рЃЊрЃљ рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ
рЃДрЃЋрЃћрЃџрЃљ рЃњрЃљрЃарЃћ рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃЊрЃљрЃДрЃћрЃюрЃЊрЃћрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃБрЃа Python-рЃерЃў:

```bash
# ­Ъћ╣ Manjaro / Linux

# рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃў рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃў
```bash 
sudo pacman -Syu python-ipykernel python-jupyter_client python-qtconsole
```
# PySide6 рЃљрЃарЃўрЃА AUR рЃърЃљрЃЎрЃћрЃбрЃў Рђћ рЃЊрЃљрЃљрЃДрЃћрЃюрЃћ yay-рЃўрЃА рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ
```bash
sudo pacman -S base-devel git yay   # рЃЌрЃБ рЃ»рЃћрЃа рЃљрЃа рЃњрЃљрЃЦрЃЋрЃА
yay -S pyside6
```
